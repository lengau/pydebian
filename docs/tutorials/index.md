# Tutorials

Tutorials take you through a series of steps to learn how to use pydebian.
They are **learning-oriented**: designed to help you get started rather than
accomplish a specific real-world task.

- [Getting started](getting-started.md) — install pydebian and make your first
  calls
- [Detecting build systems](detecting-build-systems.md) — explore debhelper's
  build system detection from Python
